Flujo de trabajo
1. Crear una muestra — en la pestaña Muestras, cargá el diseño muestral: nombre, tipo de cobertura geográfica, tabla de referencia (CSV con nivel1 / nivel2 / estrato), grupos de edad y los % de cada combinación Género×Edad×Estrato.

2. Activar la muestra — hacé clic en "Usar esta muestra" e ingresá el N objetivo. El sistema calcula automáticamente los casos por cuota.

3. Cargar datos — exportá el CSV de SurveyMonkey y cargalo en la pestaña Cargar datos. Mapeá las columnas.

4. Monitorear — el Dashboard muestra el avance de cada cuota en tiempo real con barras de progreso y alertas.

5. Firebase — para sincronización en tiempo real entre dispositivos, configurá tu proyecto Firebase pegando las credenciales al inicio del archivo HTML.
